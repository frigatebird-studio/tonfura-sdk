getProvider

  • getProvider(): Promise<TonfuraProvider>
    • Defined in api/TonfuraConfig.ts:63

    Returns an TonfuraProvider instance. Only one provider is created per Tonfura instance.

    The TonfuraProvider is a wrapper around TonfuraProvider class and has been expanded to support Tonfura's Enhanced APIs.

    Most common methods on the provider are available as top-level methods on the Tonfura instance, but the provider is exposed here to access other less-common methods.
